# clinic_web_project
Projet de groupe : application web pour la gestion d'une clinique médicale avec interfaces dédiées pour chaque utilisateur


# Projet Clinic Web - Groupe de 6

Bienvenue sur le dépôt de notre projet !! 

##  Instructions pour l'équipe
Chaque membre travaille **uniquement** dans son dossier dédié. Ne modifiez pas les fichiers des autres.

## Répartition des dossiers et tâches :

1. **membre1-rdv** farah : Gestion des RDV (Formulaire patient + Calendrier docteur).
2. **membre2-dossiers** maram : Dossiers Patients (Consultation analyses + Ajout rapports PDF).
3. **membre3-annuaire** safa: Annuaire Médical (Liste médecins + Filtres + Gestion profils).
4. **membre4-pharmacie** ons : Pharmacie Interne (Liste médicaments + Inventaire stock).
5. **membre5-blog** mohamed : Blog Santé (Articles en Cards CSS + Éditeur de texte).
6. **membre6-paiements** wissem  : Paiements (Historique factures + Dashboard revenus).

## Comment ajouter votre travail ?
1. Cliquez sur **votre dossier** ci-dessus.
2. Cliquez sur **Add file** > **Upload files**.
3. Glissez vos fichiers `.html` et `.css` dans la zone.
4. Dans la case "Commit changes", écrivez ce que vous avez fait (ex: "Ajout du formulaire").
5. Cliquez sur le bouton vert **Commit changes**.
